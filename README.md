# Little Fashion - Modern E-commerce Landing Page

A responsive, and high-performance fashion landing page built with **React.js** and **Tailwind CSS**. This project focuses on clean UI/UX, smooth animations, and a modern shopping experience.

## Features

* **Modern Hero Section:** Animated sliding images with custom pagination bubbles.
* **Interactive Tabs:** Smooth transition between Introduction, How we work, and Capabilities sections.
* **Product Gallery:** Clean grid layout with hover effects, wishlist icons, and "Add to Cart" functionality.
* **Dynamic UI:** Fully responsive design that looks great on all devices (Mobile, Tablet, Desktop).
* **Footer:** Dark-themed footer with social media integration (WhatsApp, YouTube, Instagram, X).
* **Animations:** Powered by **Framer Motion** for a premium feel.

## Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **Animations:** Framer Motion
